# Pretraining Data Card — NanoFinance 预训练数据

## 概述

本文档描述 NanoFinance 项目预训练阶段使用的数据组成、来源、处理规则与已知局限。预训练数据由英文通用（ClimbMix）、中文通用（SkyPile/Wiki）与中文金融数据混合而成。**重要声明**：预训练数据的总 Token 池与各组成部分 Token 数为 **[历史自报]**——原始训练日志不可重新获取，无法独立验证。所有不可重新获取的数据均明确标记为 `historical artifact unavailable`。本卡片不硬编码不可验证的数字，引用时请标注来源类别。

---
